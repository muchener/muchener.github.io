第3章 准备好作品

竞品分析报告

目的

思考产品本质，提高自身服务水平。

竞品分类：

  • 直接竞争类
  • 间接竞争类
  • 潜在竞争类

获得信息渠道：

  • 公司的季报、年报
  • 公司官网、产品论坛
  • 艾瑞、易观国际等数据网站的行业分析报告
  • 行业媒体的新闻、论坛、分析文章等
  • 从对方公司、相关渠道、市场、运营等部门获取对方内部信息等
  • 作为用户，使用对方产品，进行客服或技术咨询等，获取想了解的信息
  • 遵照对方产品用户（核心用户、普通用户等）进行访谈等
  • 百度指数、alexa.com、Google adpanner等

分析维度

  • 行业分析
    • 行业整体情况
    • 如产品市场占比
    • 盈利模式、代表性产品、目标用户区别等
  • 盈利模式
    • 产品盈利模式
    • 运营资金投入情况
    • 该产品是否为公司核心产品等
  • 产品技术
    • 抗压能力
    • 稳定性
    • 技术框架
  • 战略层
    • 产品定位
    • 目标用户
    • 核心需求
  • 范围层
    • 目标用户的相关需求有哪些
    • 功能
  • 结构层
    • 流程结构
  • 框架层
    • 页面布局
    • 页内组件布局
  • 表现层
    • 产品UI和UE
  • 数据分析
    • 盈利情况
    • 推广情况
    • 用户群体覆盖面
    • 市场占有率
    • 用户数量/装机量/转化率/在线时长等

分析方法

  1. SWOT（strength优势、weaknesses劣势、opportunities机会、threats威胁）
  2. 信息对比
  • 目标用户
  • 盈利模式
  • 核心功能（有哪些，差别是什么、用户满意度）
  • UI
  • UE
  1. 十字交叉法
  2. 5W1H
  • what、why、where、when、who、how

产品分析报告

层次分析法

从产品维度来考虑

递进分析法

  • 产品定位
    • 可变
  • 目标用户
  • 使用场景
  • 需求分析
    • 现状、目的、驱动力（问题）、阻力（成本、代价）
  • 产品市场分析
    • 评估因素：产品市场状况、同类产品市场竞争、产品性价比营销手段产品优势
    • 结论：产品在目前市场的竞争环境下的生存空间、产品提升市场占有率的可能性及前景
  • 产品发展前景、产品现阶段
  • 产品容量、产品变化趋势、技术基础
  • 产品易用性、功能完整性、用户体验
    • 易见、易学、易操作
  • 产品优化
    • 功能增加
    • 功能优化
    • UI优化
    • 优化顺序

个人项目总结报告

项目介绍顺序

  • 项目基本介绍
  • 目前成果进度
  • 各项基本分析：市场分析、竞品分析、用户分析、需求分析
  • 产品详细介绍：各个模块的关系、功能、解决什么问题
  • 项目重点难点介绍和分析
  • 个人收获总结

个人作品案例

基本介绍->调研->思考->设计->开发与测试
