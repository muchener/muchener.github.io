运营分类

产品运营

运营职能

互联网产品价值 = 功能 + 体验 + 用户参与价值

用户参与价值：

  • 用户的使用时间和关注给产品带来的额外价值
  • 用户在互相给对方创造和提供价值
  • 用户认可产品，愿为你贡献价值

例：微信群

  • 给群设定一个好主题（外包装）
  • 把一批跟这个主题相匹配的人拉进这个群（定位用户）
  • 明群群规
  • 制定话题，活跃气氛
  • 定期组织线上线下活动

运营手段

  1. 拉新、引流、转化。
  2. 用户维系

运营工作流程

  1. 制定策略 => 评估产品形体、数据
  2. 分解指标 => 目标拆分，根据目标完成资源向内向外的资源争取和分配，制定具体的工作计划。
  3. 执行落地 => 通过文案、活动、用户维系、BD拓展等具体手段来达成目标。
  4. 监测数据、调整方向 => 根据数据分析结果以及用户反馈，寻找潜在产品方向，制定新的运营策略。

运营与市场的区别与关联

市场部

  • 创造和管理消费者无形价值的部门
  • 瞄准扩大品牌、产品的的用户认知和提升产品的无形价值。

    我的感受：
    感觉市场是提升公司人气和产品品牌度而存在的，它是从宏观去做宣传，注重的是大量、广泛，比较虚但又不可缺少。
    而运营则是注重落实在产品的实际数据上。

例如：
三节课用户与三节课发生关系图

市场部需要做的就是广泛宣传，让更多的人了解知道三节课是做什么的，也就是图中的第一步。
而后三部则是运营重点关注的。

产品与运营

产品提供长期用户价值，运营创造短期价值+协助产品完善长期价值。

产品”烂“作为运营如何做：

  • 维护早期核心用户，不断推动产品更新迭代和打磨。
  • 用数据说明一切
  • 提出可行的解决方案，不一味抱怨

互联网的发展史

  • QQ：1999.2月上线
  • 阿里巴巴、当当：1999年
  • 淘宝：2003年
  • 微博：2009.8
  • 微信：2012

读了从互联网开始萌芽到如今的互联网时代，明朝那些事就是不折不扣的网文，后来又看到作者提到的京东客服mm的事，特地去搜了下，看了截图，感慨好久她幽默风趣的语言。

  2. 广告收入
  3. 分析用户流量，以便针对性推广
